🔥 Firebase Bağlantısı
Firebase projenizin yapılandırma kodunu girin.
1console.firebase.google.com adresine gidin, proje oluşturun.
2Proje Ayarları → Web uygulaması ekle → firebaseConfig nesnesini kopyalayın.
3Authentication → E-posta/Şifre oturum açmayı etkinleştirin.
4Firestore Database → Oluştur → Test modunda başlat.
